# References

[1] **React.js Documentation.** Official documentation used for building the component-based user interface and managing reactive application state.  
URL: [https://react.dev/](https://react.dev/)

[2] **Flask Framework Documentation.** The primary technical resource for developing the RESTful API and implementing server-side business logic and routing.  
URL: [https://flask.palletsprojects.com/](https://flask.palletsprojects.com/)

[3] **Tailwind CSS Documentation.** Utilized for the implementation of a utility-first design system and custom styling of the application's premium user interface.  
URL: [https://tailwindcss.com/docs](https://tailwindcss.com/docs)

[4] **SQLAlchemy ORM Documentation.** Reference for managing relational database models and ensuring data integrity through Object-Relational Mapping.  
URL: [https://docs.sqlalchemy.org/](https://docs.sqlalchemy.org/)

[5] **Flask-JWT-Extended Documentation.** Official resource used for implementing secure, token-based authentication and role-based access control.  
URL: [https://flask-jwt-extended.readthedocs.io/](https://flask-jwt-extended.readthedocs.io/)

[6] **Framer Motion API Reference.** Technical documentation used for creating advanced UI transitions and high-fidelity micro-animations.  
URL: [https://www.framer.com/motion/](https://www.framer.com/motion/)

[7] **MDN Web Docs (Mozilla).** Comprehensive reference for standard Web APIs, JavaScript ES6+ features, and modern HTML5/CSS3 implementation.  
URL: [https://developer.mozilla.org/](https://developer.mozilla.org/)

[8] **React Router Documentation.** Utilized for managing declarative routing and navigation within the Single Page Application (SPA) architecture.  
URL: [https://reactrouter.com/](https://reactrouter.com/)

[9] **Lucide Icons Documentation.** Reference for integrating high-quality, scalable vector icons to enhance visual communication within the UI.  
URL: [https://lucide.dev/docs](https://lucide.dev/docs)

[10] **Vite Build Tool Documentation.** Resource for configuring the development environment and optimizing the production build of the frontend application.  
URL: [https://vitejs.dev/guide/](https://vitejs.dev/guide/)
